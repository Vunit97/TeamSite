Game

Jump & Run

Ein klassiches 2D Jump & Run haben wir mit unseren eigenen Illustrationen geschaffen. Fairy World ist ein simples 2D Spiel, welches in Unity entwickelt wurde. Auf dem Weg zu seinem Ziel muss der User vier Levels überwinden. Damit er am Ende als Sieger da steht, muss dieser in der letzten Stage einen Boss bezwingen. Die hierfür verwendeten Assets wurden von unseren Designer:innen in Clip Paint Studio angefertigt. Auf persönlichen Anfragen und Anpassungen konnten unsere Designer:innen die Assets mühelos ändern und anpassen. Das Spiel ist inspiriert vom Anime Fairy Tail. Bei weiteren Anregungen meldet euch gerne bei Mariko Toelke und Kristina Nguyen, welche die Assets erstellt haben.

Game

App & Paper

Mit unserem Spiel App&Paper wollten wir eine Brücke zwischen Gesellschaftsspielen am Tisch und der digitalen Welt über das Smartphone schlagen. In dem Pen-&-Paper-Rollenspiel wird man gemeinsam in eine von einem neuartigen Virus befallene Welt geworfen und versucht in dieser zu überleben. Entstanden ist das Game durch langwieriges Planen möglicher Handlungsstränge für ein ansprechendes Storytelling und visuelle Unterstützung durch eine Karte, auf der sich die Spielenden orientieren können.

Web

Bankist Website

Um stetig unsere Fertigkeiten auf dem neusten Stand zu halten, bilden wir uns auch außerhalb der Uni weiter und üben so viel es geht. Aus diesem Grund wurde die Bankist Website geschaffen. Als Team haben wir uns wiedergefunden und haben gemeinsam an dem kleinen Projekt gearbeitet. Die Seite ist in HTML, CSS und JavaScript geschrieben und beinhaltet einen login und ein klassiches overlay von einer Überweisungsübersicht. Sobald der User sich eingeloggt hat, findet dieser eine Übersicht seines Kontos. Im eingeloggten Zustand ist es ihm möglich Geld zu überweisen, anzufodern oder das Konto zu schließen.

RDB

Bestarium Datenbank

Die Bestarium Datenbank wurde im Rahmen der Vorlesungsveranstaltung Relationale Datenbanken geschaffen und ist ein Projekt von unserem Programmierer Jimmy Cu. Es ist eine Datenbank, welche die Daten über alle Monster bzw. Kreaturen in dem Spieluniversum The Witcher 3 festhält. Wichtig für das Projekt war, dass die Datenbank in eine Webanwendung implementiert wird. Der User soll nämlich über wenige Klicks auf die Bestariumseite weitergeleitet werden und dort die Ansammlung aller Monster wiederfinden. Die Seite ist schlicht gehalten, sodass gezielt der Fokus auf die Datenbank gesetzt wurde. Die Datenbank wurde in MySQL umgesetzt.
